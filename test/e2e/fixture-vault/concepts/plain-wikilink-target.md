This note is the target of a plain `[[plain-wikilink-target]]` wikilink with no alias.

## What makes a plain wikilink plain

A plain wikilink contains only the target slug — no pipe character, no alias segment. The renderer uses the note title as the display text when the link resolves.

Understanding plain wikilinks is foundational; every more-complex form is built on top of this base case.
