Cross-directory wikilinks are common in Obsidian vaults where users maintain separate folders for concepts, daily notes, and project logs.

## Resolution strategy

The slug map is built from all notes in all ingested directories, so cross-directory resolution works as long as both directories are included in `baseDirs`.
