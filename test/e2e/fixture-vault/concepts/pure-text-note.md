---
title: Pure Text Note
tags: [concept]
---

This note contains only plain text — no wikilinks, no code fences, no images. It is the simplest possible note shape.

Obsidian vaults commonly contain notes that are pure prose journal entries or reading notes without any cross-references. The pipeline should ingest these without errors and without generating any wikilink rows.

The absence of wikilinks in the parsed content is itself a meaningful signal: this note is a leaf in the backlink graph.
