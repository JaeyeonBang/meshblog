This note exists to test aliased wikilink rendering: `[[aliased-note|see here]]`.

In Obsidian, the pipe syntax lets the author choose a display label independent of the target slug. The link still resolves to this note but the reader sees "see here" as the anchor text.

## Alias vs redirect

An alias is purely a display concern. It does not create a separate URL — only the canonical slug is addressable.
